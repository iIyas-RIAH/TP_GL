Les diagrammes de cas d'utilisation (DCU) sont des diagrammes UML utilisés pour donner une vision globale du comportement fonctionnel d'un système logiciel. 

1- installed
2- les differents element du diag. de cas d'utilisation :
   -cas d'utilisation(une fonction qu'un système exécute pour atteindre l'objectif de l'utilisateur. )
   -acteurs(représente un rôle d'un utilisateur qui interagit avec le système)
   -sous systemes (les sous-systèmes sont un type de composant stéréotypé représentant des unités)
   -relations dans les diag.
